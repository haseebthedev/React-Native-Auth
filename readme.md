### Test Task for React Native

Hi,

This is a test task for React Native by Ropstam. This project is completely build from scratch by me.

The project structure is as follows:

#### Entry Point: src

- @types
This contains all types required by modular convensions / packages.

- assets
This contains icons as form of images. We can also place fonts, image assets in this folder.

- components
This contains all individual / reusable components that I have used in this
task.

- constants
This contains constants such as colors. If I used custom fonts in this project, I would have placed all those fonts in fonts.ts.

- enums
This contains all the enums files respective of their enums.

- hooks
This contains all the hooks. Currently, I have created only one hook for auth in this task.

- screens
This contains the screens that the project is having such as Login, Homepage etc.

- services
This is the folder where I have created service controllers for fetching data from servers. Also, I have used built-in fetch module.

- utils
This is a utility folder where I have created several utilities such as validations for inputfields, storage and responsiveness etc.
